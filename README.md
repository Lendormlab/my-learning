Day1学习内容
学习解析器如何获取网页ip：
解析器先在缓存中读取，如果没有读取到，去根服务器，根服务器会告诉解析器权威服务器，权威服务器会告诉解析器去哪里找ip，ip就被带回来了，然后网页可以根据ip传输过来的内容进行渲染。
